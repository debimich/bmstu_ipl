Теорема Пифагора

Определение теоремы Пифагора:

В прямоугольном треугольнике квадрат длины гипотенузы равен сумме квадратов длин катетов.

Гипотенуза – сторона, лежащая напротив прямого угла.

Катет – одна из двух сторон, образующих прямой угол.

Формула теоремы Пифагора выглядит так:

a2+b2=c2

где a, b – катеты, c – гипотенуза.

Также верна и теорема, обратная теореме Пифагора:

Если сумма квадратов катетов двух сторон треугольника равна квадрату третьей стороны, то такой треугольник является прямоугольным.

Программа на си, вычисляющая длину гипотенузы:

#include <stdio.h>
#include <math.h>

int main() {
int a, b;
scanf_s("%d %d", &a, &b);
printf("%.2lf", sqrt(a * a + b * b));
return 0;
}
